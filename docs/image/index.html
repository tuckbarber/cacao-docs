Image

To be applied to images for general sizing and positioning.

Available classes

  • u-img: Make an image shrink to the width of its containing element if the element is smaller than the image's original width.
  • u-imgFull: Make an image scale to the width of its containing element regardless of the image's original width.

Usage


u-img (container 300px, image 100px)

u-imgFull (container 300px, image 100px)
