title: Breadcrumb
---

Provides basic setup for breadcrumb navigation.

## Available classes

* `Breadcrumb`: The core breadcrumb component.
* `Breadcrumb-link`: A link element.

## Usage

```html
<div class="Breadcrumb">
  <a class="Breadcrumb-link" href="{{url}}">Link</a>
  <a class="Breadcrumb-link" href="{{url}}">Link</a>
</div>
```
